# FairVoice — AI Hiring Bias Detector
> Audit-first hiring. A human always in the loop.

## The Problem
AI shortlists candidates before humans see a 
resume — with bias baked in.

- Amazon scrapped its AI hiring tool in 2018 
  after it systematically downgraded women's resumes
- MIT & Stanford audits found identical resumes 
  with "foreign" names score 10–15% lower
- NLP interview tools penalise non-native accents 
  even when the answer is identical
- Hiring managers trust the AI score. 
  They never see the bias.

## What FairVoice Does
1. **Side-by-side scoring** — Both candidates 
   scored against the same job description by GPT-4o
2. **Bias detection** — Score gap over 10 points 
   triggers a bias flag
3. **Human review gate** — Decision is locked until 
   a human checks 3 boxes and writes a justification
4. **Audit log** — Every decision downloads a 
   timestamped JSON record for regulatory filing
